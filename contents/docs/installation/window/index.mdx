# Install Robonito on Windows

This guide explains how to install the **Robonito** desktop application on Windows operating systems.

---

## Prerequisites

- **Operating System:** Windows 10 or later. **Must be a 64-bit architecture.**
- **Permissions:** You must have Administrator privileges to install new software on the machine.

---

## 1. Download the Installer

Download the latest Robonito setup executable directly from our secure build server:

📥 **[Download Robonito for Windows (v2.5.0)](https://robonito-prod-builds.s3.ap-south-1.amazonaws.com/Robonito-Setup-2.5.0.exe)**

---

## 2. Execute the Setup Wizard

1. Locate the downloaded `Robonito-Setup-2.5.0.exe` file in your Downloads folder.
2. Double-click the file to launch the installation wizard.

### Handling Windows SmartScreen Warnings
When running the installer on a fresh machine, Windows Defender SmartScreen may display a blue warning screen preventing the app from starting.

![Windows SmartScreen Warning](/Windows-Warning.png)

This is a standard Microsoft security prompt for applications downloaded from the web. To proceed:
1. Click the **More info** text link located below the prompt.
2. Click the **Run anyway** button that appears.

---

## 3. Finalize Installation

Follow the standard setup dialogs to complete the installation. Leaving the destination folder as its default path is highly recommended.

Once installed, launch Robonito from your Start Menu, sign in with your active account credentials, and begin recording your automation workflows.
