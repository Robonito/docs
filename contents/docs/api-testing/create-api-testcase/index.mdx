### Prerequisite

Before creating an API test case in Robonito, you need to have an API suite.  
[Learn how to create an API suite](../suites)

### Creating an API Test Case

- **Step 1**: Open the test explorer by clicking the second icon in the sidebar on the dashboard page.  
  ![Step 1](/api-testing/test-explorer-instruction.png)

- **Step 2**: In the suite list, click the action menu button for one of the API suites.  
  ![Step 2](/api-testing/api-suite-menu-item.png)

- **Step 3**: Click the **Add Test Case** button. Once the modal opens, fill in the details and create the test case.  
  ![Step 3](/api-testing/create-testcase-button.png)

- **Step 4**: After the test case is created, add a request by clicking the **Add Request** button.  
  _(A request is an entity in the API test case that contains all the details of the API request.)_  
  ![Step 4](/api-testing/create-endpoint.png)

### ✅ What's Next?

You’ve created your first API test case.
