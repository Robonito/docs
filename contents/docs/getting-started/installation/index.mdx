The Robonito Desktop App is available for Windows and Mac

## Installation Overview

Head over to [Robonito Website](https://robonito.com)

On the homepage click on the download button for your environment.

![Download Robonito](https://raw.githubusercontent.com/Robonito/docs/refs/heads/main/public/Robonito-HomePage.png)

Once downloaded it's a straightforward process of installation on either Mac or Windows. Please note that our Windows app is not yet signed so you may get a warning from Windows Defender like the screenshot below.

![Windows Warning](https://raw.githubusercontent.com/Robonito/docs/refs/heads/main/public/Windows-Warning.png)

Please Click Run anyway to allow the installation.*

Once installed you should can launch Robonito from Start Menu in Windows or Applications in Mac.

We shall explore different features of Robonito in other sections. In case you get any issues please join us on [Slack](https://join.slack.com/t/robonito/shared_invite/zt-2vzwc6wol-wJUrgT4aL9~LbhiCYwdczA)
