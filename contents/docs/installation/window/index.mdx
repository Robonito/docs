#### Installation Overview
    To install the Robonito Desktop App on your Windows, follow these steps:

1. **Visit the Robonito Website**  
   Navigate to the [Robonito Website](https://robonito.com).

2. **Download the Robonito Desktop App** 
![Download Robonito](https://raw.githubusercontent.com/Robonito/docs/refs/heads/main/public/Robonito-HomePage.png)

Click the "Download" button to retrieve the Robonito Desktop App for Windows.  
Once selected, the download will start automatically.

3. **Install the App** 
Once the download is complete, locate the file in your Downloads folder. Double-click the installer and follow the on-screen instructions to complete the installation.

(Note: our Windows app is not yet signed, so you may get a warning from Windows Defender as shown below).

![Windows Warning](https://raw.githubusercontent.com/Robonito/docs/refs/heads/main/public/Windows-Warning.png)

Please Click Run anyway to allow the installation.\*

**Launch the App**  
Once installed, you can launch Robonito from the Start Menu in Windows..
