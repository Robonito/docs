# Install Robonito on Windows

This guide explains exactly how to securely install and set up the **Robonito** desktop application on Windows operating systems.

---

## System Requirements

Before downloading and installing Robonito, please ensure your host system meets the following strict requirements:

- **Operating System:** Windows 10 or later (64-bit architecture strictly required).
- **Browser:** Google Chrome (the latest stable version is highly recommended to guarantee the best automation execution).
- **Internet Connection:** An active broadband connection is strictly required for initial installation and background updates.
- **System Permissions:** You must have proper Administrator access to install new software on your machine.

---

Click the secure link below to successfully download the latest robust version of the Robonito setup installer:

  [Download Robonito for Windows (v2.5.0)](https://robonito-prod-builds.s3.ap-south-1.amazonaws.com/Robonito-Setup-2.5.0.exe)

---

## Installation Steps

### Step 1: Download the Installer
Securely download the Robonito Windows executable installer file (`.exe`) from the designated link above.

### Step 2: Run the Setup
Double-click the downloaded `.exe` installer file to officially launch the setup wizard on your desktop.

### Step 3: Handle Windows SmartScreen (If Prompted)
When running the installer for the first time, Windows Defender SmartScreen may unexpectedly display a blue warning screen preventing the app from starting natively.

![Windows SmartScreen Warning](/Windows-Warning.png)

> **Security Note:** Robonito is completely safe and verified to install. This warning does **not** indicate malware—it is simply a standard Windows security prompt for newly published applications on the internet.

To safely bypass the prompt and continue your installation:
1. Click the **More info** text link on the warning screen.
2. Click the **Run anyway** button that appears at the bottom.

### Step 4: Complete the Setup Wizard
After successfully bypassing the prompt and launching the installer:
1. Follow the clear on-screen instructions provided natively in the setup wizard.
2. Choose your preferred installation directory (leaving it as the default file path is highly recommended).
3. Click **Install** and wait to complete the entire installation process.

---

## After Installation

Once the setup is completely finished:
- Launch Robonito directly from your **Start Menu** or the newly created desktop shortcut.
- Ensure **Google Chrome** is officially installed and updated to the latest version on your Windows system.
- Log in to your connected account. Robonito is now fully ready to record and securely execute your automated tests!
