# Installation Guide

This section seamlessly explains how to download, install, and set up the **Robonito** desktop application on Windows, macOS, and Linux systems. Follow the instructions for your specific operating system to get started with automated testing quickly.

---

## System Requirements

Before downloading and installing Robonito, please ensure that your primary host system meets the following requirements to guarantee smooth recording capabilities and the rapid execution of local test cases.

### Operating System
- **Windows:** Windows 10 or later (64-bit strictly required).
- **macOS:** macOS 11 (Big Sur) or any later version (Apple Silicon/ARM64 strictly required).
- **Linux:** Ubuntu 20.04 or later (Debian-based distributions are highly recommended).

### Hardware Requirements
- **Processor:** Dual-core CPU or higher.
- **Memory (RAM):** Minimum 8 GB (16 GB is highly recommended for optimal test automation performance).
- **Disk Space:** At least 5 GB of free hard drive space to accommodate application files, cache logs, and portable browser downloads.
