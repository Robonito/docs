# AI-Generated Assertions

> **Coming Soon:** This feature is currently under active development.

AI-Generated Assertions will allow Robonito's AI engine to automatically propose and insert validation steps as you record user flows — saving you from manually adding every assertion.

---

## How It Will Work

After each action you perform during recording (e.g., clicking a button that triggers a result), the AI engine will:
1. Analyze the current state of the UI.
2. Suggest an appropriate assertion (e.g., "The success banner should be visible").
3. Allow you to accept, reject, or modify the suggestion before it is saved to the test.

---

## Current Alternative

You can already use the [Smart Assertion](/docs/assertions/smart-assertion) tool to create AI-powered validations inline during any recording session.
