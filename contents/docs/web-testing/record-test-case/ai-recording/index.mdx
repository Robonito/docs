# AI-Assisted Recording

> **Coming Soon:** This feature is currently under active development. Check back for updates.

Robonito's AI-Assisted Recording mode is designed to augment the standard manual recorder by intelligently suggesting test steps, automatically handling dynamic UI elements, and filling in common interaction patterns that would otherwise require manual scripting.

---

## What to Expect

When available, AI-Assisted Recording will:
- **Suggest assertions** automatically based on what appears on screen after each action.
- **Handle dynamic content** like loading spinners, modals, and pop-ups without manual intervention.
- **Generate descriptive step names** automatically to keep your test readable.

---

## Current Alternative

While this feature is being finalized, you can use the existing [Smart Assertion](/docs/assertions/smart-assertion) tool during a standard recording session to achieve similar AI-driven validation.
