# Mobile Setup


If the required mobile environment is not configured, Robonito will display the message **“Mobile Setup Not Complete”**. This means that the mobile testing environment (SDKs or Emulator) is missing required components.

To continue with mobile test recording, click **Configure Environment** and complete the mobile setup configuration. Once the setup is finished, you can start recording mobile test cases.

![Target Suite ](/test-case/mobile-setup.png)

Configure Android SDK, Node.js, and Appium for mobile testing in Robonito.
The **Mobile Environment** section allows you to configure:

- Android SDK
- Node.js
- Appium

This setup prepares your system for mobile automation testing.
## Setup Required

Robonito manages isolated SDKs to ensure stability.

This setup will not affect your system's global environment.


## Environment Status

At the top of the page, you can see:

- Installation progress percentage
- Platform (e.g., Windows)
- Setup Required indicator
- **Install All** button

Click **Install All** to install all missing components.


# Environment Components

The following components must be installed for mobile testing:


##  Java Development Kit (JDK)

**Description:**  
Java runtime for Android tools.

**Status:**  
Shows `MISSING` if not installed.


##  Android Command Tools

**Description:**  
SDK Manager & AVD Manager.

**Status:**  
Shows `MISSING` if not installed.


##  Platform Tools (ADB)

**Description:**  
Android Debug Bridge.

**Status:**  
Shows `MISSING` if not installed.


##  Android Emulator

**Description:**  
Virtual Device (Pixel 9).

**Status:**  
Shows `MISSING` if not installed.


##  Node.js Runtime

**Description:**  
Isolated Node.js for Appium.

**Status:**  
Shows `MISSING` if not installed.


## Appium Server

**Description:**  
Automation server v2.12+

**Status:**  
Shows `MISSING` if not installed.


##  UiAutomator2 Driver

**Description:**  
Android automation driver.

**Status:**  
Shows `MISSING` if not installed.


# How to Install

1. Open **Robonito**
2. Go to **Mobile Setup**
3. Click **Install All**

Robonito installs all missing components automatically.
