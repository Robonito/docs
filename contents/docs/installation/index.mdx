# Installation Guide

This section explains how to download, install, and initialize the **Robonito** desktop application on Windows, macOS, and Linux systems. 

---

## Minimum System Requirements

To ensure stable test execution, your host machine should meet the following baseline hardware guidelines:

- **Processor:** Dual-core CPU or higher.
- **Memory (RAM):** 8 GB minimum (16 GB recommended for heavy parallel executions).
- **Disk Space:** 5 GB of free storage to accommodate application data, browser binaries, and dockerized Appium dependencies.
